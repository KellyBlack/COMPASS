

Overview
====================

The COMPASS program is an effort to bring together the first year 
physics courses and the first year calculus courses in the STEM
curriculum. The effort is funded by the National Science Foundation
(NSF IUSE Proposal 1431882). 

The materials available include the calculus worksheets. They can
be found at `github <https://github.com/KellyBlack/COMPASS>`_

The documentation contains the list of daily goals. The specific
URLs for each daily goal can be found within the pdf file that
includes the calculus worksheets.
